# Django Facts API

## Description
Django Facts API is a simple REST API built using Django REST Framework.  
It returns a hardcoded list of facts in JSON format and is designed for learning, assignments, and backend practice.

---

## Features
- RESTful API using Django REST Framework
- Returns JSON data
- Simple and clean structure
- Ready for cloud deployment

---

## Tech Stack
- Python
- Django
- Django REST Framework

---
